# AI Voice Command Automation Assistant

A Python-based desktop automation assistant that accepts voice commands and performs system operations.

## Features

- Voice recognition
- Text-to-speech response
- Open applications
- File and folder management
- Google search automation
- YouTube search automation
- Screenshot capture
- System information

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- SoundDevice
- OS Automation
- Web Automation

## How to Run

Install dependencies:

pip install -r requirements.txt

Run:

python main.py
